SOC Investigation Lab

# Overview

This repository contains hands-on Security Operations Center (SOC) investigations, simulating real-world cyber security incidents using tools such as Splunk and Sysmon.

Each case demonstrates the full investigation lifecycle:

- Log analysis
- Threat detection
- Incident triage
- Root cause identification
- Response recommendations


Cases Included:

Case 01 — Brute Force Attack

- Detection of multiple failed login attempts
- Analysis of authentication logs
- Identification of attacker behavior

Case 02 — Suspicious Activity

- Investigation using Splunk queries
- Identification of anomalous patterns
- Correlation of events across logs

Case 03 — Sysmon Analysis

- Endpoint monitoring using Sysmon
- Event log investigation
- Detection of potentially malicious activity


Tools Used

- Splunk
- Sysmon
- Windows Event Logs


Skills Demonstrated

- Log analysis
- Threat detection
- Incident response
- SIEM investigation
- Analytical thinking
